# **App Name**: HolaMundo

## Core Features:

- Display 'Hello World' Text: Render the classic 'Hello World' message on a clean web page.

## Style Guidelines:

- A very light color scheme, conveying cleanliness and minimalism. The primary color is a deep, desaturated blue-gray (#3D484F) chosen for a subtle yet grounding presence. The background color is a nearly-white shade with a hint of blue-gray (#F0F2F4) to provide a spacious, clean canvas. An accent color of muted aqua (#7AB3BF) offers a delicate highlight.
- All text will use 'Inter' (sans-serif) for its modern, neutral, and highly readable qualities, perfectly suiting a minimal content approach.
- A clean, simple, centrally aligned layout with ample whitespace, drawing immediate attention to the 'Hello World' message without distraction.
